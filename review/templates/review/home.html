The OSED

The Oxlip School Evaluation Dashboard (OSED) is designed to help schools complete a clear, termly evaluation of their current position across each key Ofsted area. School leaders use the Ofsted framework, the Oxlip supporting tool and their own school evidence base to decide which judgement best reflects the school’s current position in each area. These judgements are entered into the OSED app to create a simple colour-coded and numerical overview for the school.

For each evaluation area, the leader responsible records a concise summary of the evidence behind the judgement, along with the key next steps needed to improve or strengthen provision. This information can then be used to support the school’s self-evaluation and can be extracted for use in SEF-style reporting. The school overview can be shared with governors, while the combined Trust dashboard provides trustees with a wider picture across schools.

In addition to the termly overview, each evaluation area will receive a more detailed review at least once a year. During this in-depth review, leaders assess individual framework statements, marking whether each criterion applies and adding justification. Where all expected standard criteria are met, leaders can consider whether the strong standard applies; where criteria are not met, the school is guided towards the areas that need attention.
