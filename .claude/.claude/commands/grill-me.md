description: Ask clarifying questions before making any changes
disable-model-invocation: true
---

Before writing any code or making any changes, ask targeted questions to fully understand the requirements. Ask about:

- What exactly needs to change and why
- Constraints, patterns or conventions to follow
- Edge cases to handle
- What success looks like
- Anything that could break elsewhere

Do not write any code or make any changes until all questions have been answered.
