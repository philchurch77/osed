Import users from CSV

Upload a CSV file to bulk-create user accounts and link them to schools. New users are created without a password — they will log in using Microsoft sign-in with their work email address.

Expected CSV format:

email,school
john.smith@example.com,Mendlesham Primary School
jane.doe@example.com,Some Other School
  • A header row is required with the column names email and school.
  • School names must match exactly (case-insensitive) as they appear in the database.
  • You can have multiple rows for the same email to assign a user to multiple schools.
  • If a user with that email already exists, their account is updated (school added).
